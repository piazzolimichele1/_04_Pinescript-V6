# Pinescript-V6

Custom TradingView tools written in Pine Script v6, split into two areas:
indicators and strategies. Every tool sits in its own folder, with a versioned
file and its own readme.

## Indicators
Tools for reading the chart, market structure and context.
- [01] Market structure 2 candles
- [02] Drs_riskmanagement
- [03] Engulfing candle pattern
- [04] Quortersprice
- [05] Tradingsessions
- [06] Support&Resistance
- [07] Tradechecklist
- [08] ZigZag

## Strategies
Backtestable strategies with automated signals, one logic each.
- [01] Ichimoku
- [02] Engulfing marketstructure
- [03] Engulfing quorters
- [04] Engulfing SMA
- [05] Ichimoku Quorters
- [06] Range breakout with quorters
